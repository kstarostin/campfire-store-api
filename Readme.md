# Campfire Store API

WIP...

## Project structure

The main goal of this project was to build a webshop API using **Node.js** with **Express** framework.

### Build

Navigate to the project root and install the project dependencies with the command:

```
 npm install
```

To start the app, use:

```
 npm start
```

### Assets
